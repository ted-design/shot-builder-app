# Product Editor IA Options (Pre-Implementation)

These static mockups explore product-editor IA upgrades with stronger classification controls and faster colorway authoring.

- `option-1-taxonomy-rail.html`  
  Split editor with a managed taxonomy rail and quick colorway bulk-add.
- `option-2-guided-flow.html`  
  Guided three-step editor (Identity → Classification → Colorways) with progress cues.
- `option-3-table-studio.html`  
  Table-first colorway studio focused on dense, high-throughput SKU editing.

All options are desktop-first and include mobile-safe layout behavior for review.
