<product_requirements>
The Unbound Merino Shot Builder app requires a comprehensive refactor and build-out. Key features include:
1.  **Authentication & Role Management**: Google Sign-In and passwordless email links, Firebase custom claims for RBAC, and organization scoping.
2.  **Data Model**: Firestore collections and security rules, with RBAC enforcement.
3.  **Search**: External search service (initially Typesense) with faceted search UI and card tree drill-down.
4.  **Core Modules**: Projects, shots (with Kanban planner), products, talent, locations, and pull requests, including state transitions and event logs.
5.  **Warehouse Workflow**: Generate pull requests, allow edits/quantity adjustments, capture substitutions, and handle approval.
6.  **Technology Stack**: React with Vite, TanStack Query for state management, React Hook Form + Zod for forms, Tailwind/shadcn/ui for design. Emphasize keyboard shortcuts, accessibility, and responsive UX.
7.  **Testing**: Automated tests, Firestore rules validation with Firebase Emulator Suite, and CI setup.
8.  **New Feature Request**: PDF exports of selected shots with customization (field selection) and a PDF summary of inventory/product pulls.
</product_requirements>
