Design Notes (Phase 2b)

  • Item count: Org-level has 5 items (Dashboard, Products, Library, Settings, Admin). Project-scoped has 8 items (Back, Shots, Pulls, Assets, Call Sheet, Products, Library, Settings). Down from 12+ in the current sidebar.
  • No duplication: The "Global" section label is removed. Org items (Products, Library) appear below a thin divider in project context without a redundant heading.
  • Departments + Tags: Removed from primary nav. Departments → Project Settings > Team tab. Tags → accessible via Cmd+K or Shot filters. This removes 2 items that most users never navigate to directly.
  • "All People" group dissolved: Talent and Crew live under Library. The project-level "Assets" item covers project-scoped people/resources assignment.
  • Collapsed state: Icon-only (64px) with hover tooltips. No labels. Toggle with [ key.
  • Hover expand: Collapsed sidebar expands on hover with intentional delays (160ms open, 140ms close, 300ms min duration) to prevent janky flyover.
  • Project header: Shows project name + chevron dropdown for switching. Replaces the current FolderOpen + MoreVertical pattern with a cleaner single click-target.
  • User section: Pinned to bottom. Avatar + name + sign-out. Collapses to avatar-only.
