Design Notes (Phase 2d — Cmd+K Palette)

  • Builds on existing: Legacy SearchCommand.jsx already uses cmdk with search, actions, nav, recent searches, keyboard shortcuts. vNext refines the IA and adds richer metadata.
  • Search matches highlighted: match highlighting in results for scanability.
  • Result metadata: Status badges on shots, style numbers on products, agency on talent. Helps distinguish similar-named items.
  • "go " prefix: Typing "go " switches to navigation mode showing all destinations with Alt+key shortcuts. Matches simplified IA — no Tags or Departments nav items.
  • Mobile: Full-screen takeover (not floating dialog). Back arrow instead of ESC. Larger touch targets (48px rows). Triggered by search icon in top bar.
  • Empty state: Shows "Create Shot" / "Create Product" CTAs so users always have a next action.
  • Project context: Actions like "Create Shot" show the active project name. Nav items show project context where applicable.
