Architecture Summary

Data Flow

ExportBuilderPage
  |
  +-- ExportDataProvider (React Context)
  |     |-- useProject(projectId)         -> project metadata
  |     |-- useShots()                    -> project shots (real-time)
  |     |-- useProductFamilies()          -> org products
  |     |-- usePulls()                    -> project pulls
  |     |-- useCrewLibrary()              -> org crew
  |     |-- useTalentLibrary()            -> org talent
  |     +-- returns: { project, shots, products, pulls, crew, talent, loading }
  |
  +-- getDynamicVariables(project, shots, products)
  |     -> resolves {{projectName}}, {{shotCount}}, etc.
  |
  +-- DocumentPreview
  |     +-- BlockRenderer (for each block)
  |           +-- ShotGridBlockView     (reads shots from context)
  |           +-- ShotDetailBlockView   (reads single shot from context)
  |           +-- ProductTableBlockView (reads products from context)
  |           +-- PullSheetBlockView    (reads pulls from context)
  |           +-- CrewListBlockView     (reads crew from context)
  |           +-- TextBlockView         (reads variables for chips)
  |           +-- ImageBlockView        (no external data)
  |           +-- DividerBlockView      (no external data)
  |
  +-- PDF Generation (on Export click)
        +-- generateExportPdf()
              1. resolveVariables()     (text token replacement)
              2. resolveExportImages()  (Firebase Storage -> data URLs)
              3. render ExportPdfDocument (per-block PDF components)
              4. pdf().toBlob()         (@react-pdf/renderer)
              5. downloadBlob()         (triggers browser download)
