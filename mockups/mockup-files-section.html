Design rationale

Identity images moved to Colorways tab. Files section shows documents with inline previews for quick scanning — no download needed to see what you have. Images render as thumbnails, PDFs show a first-page preview, and non-previewable formats display a clear type icon. Grid view is the default for visual scanning; list view is available for metadata-dense workflows. The file type badge on each card provides instant format recognition.
