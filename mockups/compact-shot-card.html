Field Specifications

# Field Always Visible Behavior
1 Drag Handle Yes Initiates drag-and-drop
2 Thumbnail Yes Square aspect ratio, falls back to placeholder
3 Shot Number Yes Format: Lane#-Seq# (e.g., 1-01)
3 Status Dot Yes Color-coded indicator with tooltip
4 Shot Name Yes Truncates with ellipsis
5 Talent Yes Comma-separated, shows "No Talent" if empty
6 Products Yes Comma-separated, truncates
7 Time Yes Shows scheduled time or "--" if none
8 Actions Yes Status dropdown + Edit button
